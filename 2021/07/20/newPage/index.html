Hexo+gitHub搭建个人博客

准备工作：
1.注册github账号；
2.node、npm熟悉相关知识；
3.安装git

gitHub部分：

1.注册github账号；
2.需新建一个仓库，以xxxx.github.io结尾的心仓库；
3.配置SSH key

配置SSH key相关：
ssh-keygen -t rsa -C “邮件地址”

然后连续3次回车，最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容，打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：
验证配置成功：
ssh -T git@github.com # 注意邮箱地址不用改
如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，然后会看到：
Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access.
还需配置一些你的用户名跟邮箱

git config –global user.name “liuxianan”// 你的github用户名，非昵称

git config –global user.email “xxx@qq.com“// 填写你的github注册邮箱

这样配置可以直接推送到github上，可以不输入账户密码。

hexo
hexo是一个简单，快速，强大基于github可以发布博客的一个工具，支持Markdown格式，有众多优秀插件和主题。
官网： http://hexo.io

安装：

npm install -g hexo

初始化：
回到你的根目录 进行初始化项目 npm init，会生成一些目录文件：
生成：

hexo -g

启本地服务器，会有http://localhost:4000

hexo s

配置hexo文件目录下的_config.yml中有关deploy的部分：
deploy:
type: git
repo: git@github.com:xiaoowan/xiaoowan.github.io
branch: main
注意：repo 不能写成https://github.com/liuxianan/liuxianan.github.io.git或者SSH的SSH://github.com/liuxianan/liuxianan.github.io.git，不然会推送失败

推送GitHub:

hexo d

需要更换主题：
从github中下载主题文件：
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
将文件放入theme文件目录下，修改_config.yml中的theme: landscape改为theme: yilia，然后重新执行hexo g来重新生成。

查看生成的博客：https://xiaoowan.github.io/
